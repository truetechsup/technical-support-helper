# Technical Support Helper - Приложение поиска ошибок

Простое веб-приложение для поиска информации об ошибках. Разработано для сотрудников технической поддержки.

## Структура проекта

```
app/
├── server.py              # Python HTTP сервер (API)
├── database.py            # Работа с БД
├── init_db.py             # Инициализация БД с тестовыми данными
├── templates/
│   └── index.html         # HTML страница
├── static/
│   └── style.css          # CSS стили
├── nginx/
│   └── nginx.conf         # Конфигурация nginx
├── Dockerfile.nginx       # Образ для nginx
├── Dockerfile.python      # Образ для Python
├── docker-compose.yml     # Оркестрация контейнеров
└── requirements.txt       # Зависимости
```

## Технологии

- **Python 3.11** - Backend API (стандартная библиотека)
- **Nginx** - Веб-сервер и reverse proxy
- **SQLite** - База данных
- **Docker** - Контейнеризация

## Развертывание

Этот код предназначен для развертывания на сервере. См. инструкции в основной директории проекта.

## Примечание

База данных хранится отдельно от кода приложения и не входит в этот репозиторий.
